ANIMATION INFO:

Direct X model format:
Idle1 = 1 to 507
Bye = 508 to 585
Run = 586 to 610
Search = 611 to 700
Suspect = 701 to 784
Death = 784 to 807


FBX model format:

For fbx model each animation is included in a separate take. Just call the name in the animation function.

Idle1
Bye
Run
Search
Suspect
Death

Default framerate is 24 fps
